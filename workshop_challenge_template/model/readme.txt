-------- Model building folder guide and templates


1 - store_data contains a csv file separated by ";" that will be used for model building. In addition, you can store there new data to be predicted with the same structure
2 - model-build_template.R contains the structure of the exercise of building and exporting your prediction model as a .RData file. Use your chosen data and your own methods to build up your ensamble of models
3 - model_file_example.RData is the format to deliver a trained machine to be used in predictions
