WORKSHOP CHALLENGE

1 - www folder contains a picture that you might want to display in the sidebar of the shiny app
2 - app_prototype.R contains a structure for a shiny app where prediction models can be used (Do not run, it is not complete)
3 - app_template.R contains a template that can be completed. You can work out your app on this template by completing the missing parts and commands or create your own from scratch using the app_prototype.R
4 - model folder: See readme.txt inside the folder
